Rubik cube values are in a array:

# Array Structure:
    8 | 7 | 6
    5 | 4 | 3
    2 | 1 | 0

This array values are the values on the cube face like for front array the color is orange.
it have 9 values and all these 9 values are handled by the array of 0-8 index.
different array values are uses for the Top, front, left, right, back, bottom.

# Opposite colors

(front) Orange ->  Opposite color is  -> Red     (back)
(top)   Blue   ->  Opposite color is  -> Green   (bottom)
(left)  Yellow ->  Opposite color is  -> White   (right)
